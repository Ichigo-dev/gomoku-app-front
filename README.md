# 五目並べ フロント部分
## 機能
- 五目並べを2人でできる
- 一人でもCPU相手に対戦できる
- ゲーム時に登録する名前を用いて勝利数ランキングを表示
## 環境構築(Ubuntuの場合)
```
$ sudo apt-get install nodejs
$ npm i -g yarn
$ yarn install
$ yarn start
```
